# Jumpers Heaven Statistics - Assets Submodule

This repository contains the **assets** used by the [Jumpers Heaven Statistics](https://github.com/Moamal-2000/jumpers-heaven-statistics) website.  
It is intended to be included as a **Git submodule** in the main project.

## Main Project

- **GitHub Repository:** [Jumpers Heaven Statistics](https://github.com/Moamal-2000/jumpers-heaven-statistics)  
- **Live Site:** [https://stats.jumpersheaven.com](https://stats.jumpersheaven.com)

## Usage

To include this repository as a submodule in your project:

```bash
git submodule add https://github.com/Moamal-2000/jumpers-heaven-statistics-assets assets
git submodule update --init --recursive
```
